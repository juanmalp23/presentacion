Tipos de Hosting
Ventajas e inconvenientes
Hosting gratuitos
Un alojamiento web gratuito es un servicio que permite a los usuarios publicar sitios web, y en algunos casos tener servicio de correo electrónico o base de datos de manera gratuita

Hosting compartidos

El alojamiento compartido es, por definición, un modelo de hosting en el que un conjunto de páginas web tienen acceso compartido a un conjunto de recursos en el servidor.
Hosting para revendedores
El alojamiento revendedor es un tipo de alojamiento web mediante el cual un plan de alojamiento dentro de un servidor permite a una cuenta revender alojamiento dentro del mismo plan
Hosting VPS
Un servidor virtual privado es un método de particionar un servidor físico en varios servidores virtuales de tal forma que todo funcione como si se estuviese ejecutando en una única máquina
Hosting dedicado
Un servidor dedicado es un equipo informático físico que destina todos sus recursos a proporcionar información y atender las peticiones de otro ordenador que ha contratado sus servicios.
Hosting en la Nube o Cloud Hosting
Cloud hosting o servidor en la nube es un tipo de alojamiento web que utiliza múltiples servidores diferentes para equilibrar la carga y maximizar el tiempo de actividad
